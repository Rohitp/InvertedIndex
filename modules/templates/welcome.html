So you're here. What next?

Try the insert end point

/insert

METHOD: POST

Input: Any object which follows the JSON/ Dict format

Returns {"code": 1, "message": "INSERTED"} on success

Or try the search end point

/search?q=query

METHOD: GET

q: url encoded query of search terms

use on:term to search in particular field. E.g to search on meta fields on:meta

use not:term to eliminate it from the search
